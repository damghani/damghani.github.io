# Quick Start Guide - Number Blocks Learning Adventure

## 🚀 Getting Started in 30 Seconds

### Step 1: Open the Game
Double-click `index.html` or open it in any web browser.

### Step 2: Start Playing
1. Click **"Start Learning"**
2. Choose a topic (try **Addition** first!)
3. Click **Level 1**
4. Answer 5 questions
5. Earn stars! ⭐⭐⭐

## 🎮 Controls

- **Mouse/Trackpad**: Click and drag
- **Touch Screen**: Tap and swipe
- **Keyboard**: Tab + Enter to navigate

## 🎯 Tips for Success

### For Kids
1. **Read carefully**: Look at the pictures and numbers
2. **Take your time**: There's no rush!
3. **Use the blocks**: They help you visualize the math
4. **Try again**: Wrong answers help you learn
5. **Collect stars**: See how many you can earn!

### For Parents/Teachers
1. **Monitor progress**: Check "View Progress" for analytics
2. **Focus on weak areas**: Game recommends topics that need practice
3. **Celebrate achievements**: Praise effort, not just correct answers
4. **Short sessions**: 10-15 minutes per day is ideal
5. **Build streaks**: Daily practice builds habits

## ❓ Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Game won't load | Enable JavaScript in browser |
| Progress not saving | Don't use Incognito/Private mode |
| Blocks won't drag | Try clicking/tapping and holding |
| Sounds not working | Check Settings → Sound toggle |
| Page is laggy | Enable "Reduce Motion" in Settings |
